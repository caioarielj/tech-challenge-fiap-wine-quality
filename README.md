# Tech Challenge FIAP - Classificação da Qualidade dos Vinhos

## Objetivo

Este projeto foi desenvolvido para o Tech Challenge da Pós-Tech FIAP em Data Analytics.

O objetivo é prever a qualidade de vinhos utilizando características físico-químicas e modelos de Machine Learning.

---

## Dataset

Wine Quality Dataset

1143 amostras

11 características físico-químicas

Classificação binária:

- Classe 0 → qualidade < 7
- Classe 1 → qualidade ≥ 7

---

## Modelos Avaliados

- Regressão Logística
- Regressão Logística com Oversampling
- KNN
- Random Forest

---

## Melhor Modelo

Random Forest

Accuracy: 92,14%

---

## Principais Variáveis

- Alcohol
- Sulphates
- Volatile Acidity
